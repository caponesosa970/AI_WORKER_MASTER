# Codex to ChatGPT Returns

This folder contains sanitized Markdown-only coordination packets from Codex to ChatGPT.

## Safe packet naming

Use:

`YYYYMMDD_<LANE_ID>_<RETURN_TYPE>.md`

Example:

`20260721_TEST_BRAIN_ARCHIVE_ALIGNMENT_CODEX_RETURN.md`

## Allowed Content

- Sanitized implementation summaries.
- Public-safe file lists.
- SHA256 values.
- Verification summaries.
- Open blockers and required controller decisions.

## Forbidden Content

Do not store XML, credentials, API keys, inherited key patterns, private URLs, phone numbers, messages, workbook rows, screenshots, runlogs, or private artifact bytes.
