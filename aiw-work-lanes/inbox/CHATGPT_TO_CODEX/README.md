# ChatGPT to Codex Inbox

This folder contains sanitized Markdown-only coordination packets from ChatGPT to Codex.

## Safe packet naming

Use:

`YYYYMMDD_<LANE_ID>_<SHORT_PURPOSE>.md`

Example:

`20260721_TEST_BRAIN_ARCHIVE_ALIGNMENT_SECTION12_PROMOTION_REQUEST.md`

## Allowed Content

- Sanitized instructions.
- Public-safe filenames.
- SHA256 values.
- Non-private status summaries.
- Explicit lane ID and requested action.

## Forbidden Content

Do not store XML, credentials, API keys, inherited key patterns, private URLs, phone numbers, messages, workbook rows, screenshots, runlogs, or private artifact bytes.
